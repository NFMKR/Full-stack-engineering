# 登录页工程化开发规范（Web B端项目）

## 一、页面主题
登录页风格：简约、专业、干净、现代、轻盈
整体视觉：浅灰背景 + 白色卡片 + 柔和阴影 + 12px 大圆角

## 二、背景最优方案
- 使用全屏渐变背景或纯色背景，不使用复杂图片
- 背景必须铺满全屏，不重复、不拉伸
- CSS 使用变量：--login-bg / --login-bg-gradient

## 三、登录卡片（最核心）
- 卡片宽度固定 440px，小屏幕自动变为 92% 宽度
- 卡片内边距 32px，手机端 24px
- 卡片圆角：12px（专业名称：border-radius）
- 卡片阴影：轻柔大阴影，高级不厚重
- 卡片必须在页面 垂直水平居中

## 四、标题与内容位置
- 标题放在卡片顶部居中
- 标题下方间距 24px
- 表单垂直排列，项与项之间间距 16px
- 输入框高度 40px，按钮高度 44px，按钮全屏宽度

## 五、布局实现方式
- 父容器：flex + justify-center + align-items + 100vh
- 卡片：固定宽度 + 内边距 + 圆角 + 阴影
- 响应式：自动适配手机、平板、电脑

## 六、样式必须使用变量
- 背景：--login-bg
- 卡片宽度：--login-card-width
- 卡片内边距：--login-card-padding
- 卡片圆角：--login-card-radius
- 卡片阴影：--login-card-shadow
- 表单间距：--login-form-gap
