
# [项目名称] 核心设计指南 (Design Core)

## 1\. 核心定调 (The Vibe)

  * **关键词**：[例如：工业黑石、数据极光、精密、克制]
  * **视觉隐喻**：[例如：黑暗工厂中的蓝色激光扫描]
  * **核心体验**：[例如：重力感滚动、微阻尼交互]
  * **适配策略**：以 **1440×900** (桌面标准) 为设计基准，向下兼容平板与手机，确保手机端也功能与广告等适配正常，向上适配大屏。

## 2\. 视觉规范 (Visual Specs)

### A. 色彩体系 (CSS Variables)

*不要使用纯黑 (\#000) 和 纯白 (\#FFF)*

```css
:root {
    /* 背景：深邃而非死黑 */
    --bg-core: #050505;
    --bg-sub:  #0F1115;

    /* 强调色：AI与科技感 (极光色) */
    --accent-primary: #2E5CFF; /* 工业蓝 */
    --accent-glow:    #00F0FF; /* 赛博青 (用于发光) */

    /* 文字：层级分明 */
    --text-main: #F3F4F6; /* 标题 (95%白) */
    --text-body: #9CA3AF; /* 正文 (60%白) */
    --text-dim:  #4B5563; /* 辅助 (30%白) */
}
```

### B. 排版与字体 (Typography)

  * **字体家族**：`Inter` (英) + `思源黑体` (中)
  * **字重策略**：标题 `700` / 正文 `400` / 数据 `300`
  * **响应式字号**：
      * Display: `clamp(3rem, 8vw, 6rem)`
      * Body: `1rem` (行高 1.6)

### C. 材质与光影 (Texture)

  * **玻璃拟态**：`backdrop-filter: blur(12px); bg: rgba(255,255,255,0.03); border: 1px solid rgba(255,255,255,0.05);`
  * **环境光**：`filter: blur(120px); opacity: 0.15;`
  * **噪点质感**：叠加 3% 透明度的 Noise SVG，消除数字廉价感。

### D. 图标样式（Icon）

  * **风格**：线性图标（stroke-width: 2），简洁统一，无多余装饰
  * **尺寸**：导航图标：20×20px  按钮图标：18×18px  表单 / 状态图标：16×16px  数据指标图标：24×24px
  * **颜色**：默认与当前文本颜色一致，强调图标使用 --brand-accent 或对应状态色

## 3\. 布局与空间 (Layout & Space)

  * **网格系统**：12 列布局，最大宽 1440px。
  * **呼吸感 (Spacing)**：
      * Section 间距：`160px` (大留白)
      * 组件内间距：`32px` / `64px`
  * **圆角 (Radius)**：
      * 卡片：`4px` (硬朗工业感) 或 `24px` (现代柔和感) —— *[本项目选其一]*

## 4\. 动效编排 (Motion Choreography)

### A. 滚动体感 (Scroll)

  * **引擎**：Lenis
  * **参数**：`Duration: 1.2s`, `Easing: 1.001 - 2^(-10*t)` (指数衰减，丝滑感)

### B. 进场动画 (Entrance)

  * **原则**：错峰出现 (Stagger)
  * **参数**：`y: 30px` -\> `0`, `opacity: 0` -\> `1`, `duration: 1s`, `ease: power3.out`

### C. 微交互 (Micro-interactions)

  * **Hover**：不改变位置，只改变光影 (`box-shadow` Glow) 或 透明度。
  * **光标**：自定义跟随圆点，遇见可点击元素磁吸放大。

## 5\. 工程底线 (Tech Constraints)

  * **图片**：全部 WebP 格式，大图 TinyPNG 压缩。
  * **视频**：无音轨 WebM，比特率 \< 2Mbps。
  * **性能**：Lighthouse Performance \> 90，FCP \< 1.0s。
  * **兼容**：Flex gap 降级处理，不支持 WebGL 时回退到静态图。
