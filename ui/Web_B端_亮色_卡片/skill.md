# 项目主题与样式使用指引

## 项目主题风格
本项目为 B 端 Web 系统，整体风格定位：
现代简约、干净舒适、专业克制、高可读性
视觉基调：浅灰背景 + 白色柔和卡片 + 细浅边线 + 品牌红点缀
整体视觉轻盈、透气、不拥挤、不刺眼。

## 统一样式标准（必须严格遵守）
所有页面、组件、布局、文字、颜色必须使用 style_global.css 内定义的变量，禁止直接写死色值、尺寸、间距。

## 样式文件使用规则
1. 页面背景统一使用 --bg-page
2. 卡片、面板、弹窗背景统一使用 --bg-card
3. 所有边框、分割线统一使用 --border-1
4. 圆角统一使用：
   按钮/输入框：--radius-sm
   卡片/面板：--radius-md
5. 间距必须使用 8px 栅格变量：--spacing-4 / 8 / 12 / 16 / 20 / 24 / 32
6. 文字颜色层级：
   标题：--text-primary
   正文：--text-normal
   次要说明：--text-secondary
7. 字体层级：
   辅助文字：--fs-xs
   正文/按钮：--fs-sm
   卡片标题/导航：--fs-md
8. 品牌色仅用于强调、选中状态、主按钮，禁止大面积使用
9. 图标统一规范：24×24 网格、线性、2px 线条、2px 圆角、单色、fill="currentColor"

## AI 生成代码要求
1. 所有样式必须引用 CSS 变量，不允许写死数值
2. 布局遵循 8px 栅格系统
3. 保持风格统一、干净、简约、舒适
4. 禁止使用花哨效果、厚重阴影、强对比色
5. 保持页面层级清晰、交互克制
