# 左侧边栏 工程化开发规范（B端 Web 项目通用）

## 一、整体风格
简约、干净、模块化、高可用
白色背景 + 浅灰右边框 + 柔和右侧阴影
选中/hover 使用品牌浅红背景，文字红色高亮

## 二、结构最优解
1. 侧边栏固定在左侧，宽度 220px，收起宽度 64px
2. 顶部放 Logo + 项目名称（高度 56px）
3. 中间是导航菜单（滚动区域）
4. 底部可放用户信息 / 版本号 / 收起按钮
5. 整体高度撑满视口 100vh

## 三、布局与定位
- 固定定位：position: fixed
- left: 0; top: 0; height: 100vh
- z-index: 99; 确保不被内容覆盖
- 右侧主内容自动留出侧边栏宽度

## 四、导航菜单项（最优标准）
- 高度统一 44px
- 圆角 6px
- 水平居中对齐
- 左侧图标 20px + 右侧文字
- 图标与文字间距 8px
- 菜单项之间间距 4px
- 左右内边距 12px

## 五、状态样式
- 默认：文字 #374151，背景透明
- hover：浅红背景 #fef2f2
- 选中：浅红背景 + 文字品牌红
- 禁止使用深色、粗边框、强对比色

## 六、视觉细节
- 右侧边框 1px solid #e5e7eb
- 右侧轻微阴影，提升层级
- 右上角、右下角圆角 8px
- 内容区域内边距 12px
- 滚动条隐藏或轻量化

## 七、响应式
- 桌面端：正常展开 220px
- 平板/小屏：自动收缩 64px（仅图标）
- 移动端：自动隐藏，通过按钮呼出
