# 系统全局字体工程化规范（最优解）

## 一、全局字体
统一使用系统无衬线字体，跨平台最清晰、最舒适、最稳定。
字体变量：--font-family

## 二、字号体系（固定 8 档，禁止自定义）
12px：标签、辅助文字、提示
13px：表格文字、描述文字、小字文本
14px：系统标准字号，正文、按钮、输入框、列表（最常用）
15px：模块小标题、重要说明
16px：卡片标题、侧边栏导航、一级菜单
18px：页面标题
20px：内容区主标题
24px：登录页、首页大标题

## 三、字重规范（只使用 3 个，保持统一）
400：常规正文、描述、按钮文字
500：标题、导航、重要文字（最常用）
600：主标题、强调标题、突出文字

不使用 700 粗体，避免生硬刺眼。

## 四、行高规范（保证阅读最舒适）
辅助文字：1.3
普通文字：1.4
正文内容：1.5（最通用）
标题：1.6

## 五、颜色规范（必须使用变量）
标题：--text-primary
正文：--text-normal
次要文字：--text-secondary
失效/禁用：--text-disabled

## 六、使用规则
1. 整个系统所有文字必须使用以上变量
2. 禁止自定义字号、字重、行高
3. 标题层级必须由大到小，保持视觉清晰
4. 正文统一 14px，500 字重用于标题，400 用于正文
5. 保持清晰、克制、舒适、专业
