# 🧭 **企业级官网（Corporate Professional Website）设计指南文档（Design Guideline）**

---

## **1. 品牌定位（Brand Positioning）**

该设计指南适用于企业级、大公司、商务型官网。
品牌视觉价值需体现：

* **可信赖（Trustworthy）**
* **稳重（Stable）**
* **专业（Professional）**
* **严谨（Formal）**
* **国际化（Global）**

整体风格更克制，避免过度炫酷，注重内容、信息结构清晰、阅读舒适。

---

## **2. 视觉基调（Visual Tone）**

核心视觉关键词：

* 中性、理性表达
* 清洁的布局
* 极高的留白比例
* 统一的模块逻辑
* 清晰的层次（Hierarchy）
* 内容作为主角，而非炫技视觉

> **大公司官网的视觉目标：稳健、有说服力、有秩序。**

常见视觉语言：

* 细线条
* 轻柔阴影
* 板式结构（Sections）
* 稳定配色（深蓝/深灰/品牌色点缀）

---

## **3. 色彩系统（Color System）**

企业官网的配色强调稳定、中性。

### **3.1 主品牌色（Primary Color）**

常见选项：

* 深蓝：`#003A8C`（最强企业专业色）
* 商务蓝：`#165DFF`
* 品牌深色：根据公司 VI 定义

用途：

* Logo
* 重点 CTA
* 标题强调
* 链接强调

### **3.2 辅助色（Secondary Colors）**

用于视觉区分、图表、轻度点缀：

* 灰蓝色：`#2F54EB`
* 企业灰绿：`#13A8A8`
* 商务金色（少量使用）：`#D4A017`

### **3.3 中性色（Neutral Palette）**

专业官网的核心基调：

| Token    | Hex     | 用途        |
| -------- | ------- | --------- |
| Gray-900 | #0A0A0A | 顶级标题、深色文本 |
| Gray-700 | #333333 | 正文主文本     |
| Gray-500 | #666666 | 次要信息      |
| Gray-300 | #D9D9D9 | 边框、分割线    |
| Gray-200 | #F2F2F2 | 模块背景      |
| Gray-100 | #FAFAFA | 页面背景      |

### **3.4 功能色（Semantic Colors）**

* 成功：#52C41A
* 警告：#FAAD14
* 错误：#FF4D4F
* 信息：#1677FF

---

## **4. 字体（Typography System）**

企业级官网字体强调沉稳与易读性。

### **4.1 中文字体**

* `Source Han Sans SC` / `思源黑体`
* 或 `Alibaba PuHuiTi`

### **4.2 英文字体**

* `Inter`
* `Helvetica Neue`
* `Roboto`

### **4.3 字体层级**

| 类型         | 字号      | 行高  | 用途      |
| ---------- | ------- | --- | ------- |
| H1         | 48–60px | 1.2 | 企业首页大标题 |
| H2         | 32–40px | 1.3 | 板块标题    |
| H3         | 24–28px | 1.4 | 小标题     |
| Subheading | 20px    | 1.5 | 说明性副标题  |
| Body L     | 18px    | 1.6 | 强调内容    |
| Body       | 16px    | 1.7 | 标准正文    |
| Caption    | 14px    | 1.6 | 注释      |

字体风格：
**严谨、少装饰、少夸张、对称且中性。**

---

## **5. 布局（Layout System）**

企业官网推荐：

* **12 列网格（Desktop）**
* 最大页面宽度：1440px
* 栅格间距：24px

移动端：4 列网格

布局特性：

* 左右对称
* 块状排版
* 大量留白
* 标题 → 文案 → CTA 的节奏明确
* 信息结构清晰、易阅读

---

## **6. 组件设计（Components）**

企业官网组件特点：

* 圆角较小（4px–8px）
* 边框为 1px 灰线
* 阴影非常轻：`0 2px 8px rgba(0,0,0,0.06)`
* 风格稳重克制

---

### **6.1 按钮（Button）**

**主按钮（Primary）**

```
背景：主色（深蓝）
文字：白色
圆角：6px
高度：44px
字体：16px Medium
Hover：略微增加亮度 / 阴影轻微增强
```

**次按钮（Secondary）**

```
背景：白色
边框：1px 主色
文字：主色
```

**纯文字按钮（Text Button）**

```
颜色：主色
下划线 / hover：颜色加深
```

---

### **6.2 卡片（Card）**

```
背景：白色
边框：1px #E5E5E5
圆角：8px
阴影：轻微（非科技感）
标题字体：24px
内容行高大，保持可读性
```

---

### **6.3 导航栏（Header & Navbar）**

企业官网头部通常非常“稳”：

* 左 Logo
* 右横向导航
* 光泽少
* 纯白背景或深色背景
* 黑灰文字
* Hover：下划线或浅色背景框

---

### **6.4 页脚（Footer）**

企业级 footer 更内容化：

* 公司简介
* 产品列表
* 合作伙伴
* 联系方式
* 版权 & ICP 备案（中国）
* 地图/二维码

---

## **7. 图形元素（Visual Elements）**

专业官网更注重真实感和可靠性：

* 真实场景照片
* 企业办公环境
* 合作伙伴 logo 墙
* 奖项/资质证书
* 实拍团队照片
* 简洁图标（线性）
* 内容模块不使用复杂科技背景

---

## **8. 动效（Motion System）**

企业级官网的动效特点：

* 轻（Opacity + Transform）
* 不夸张
* 下滑 reveal（分段淡入）
* 按钮 hover 切换背景色
* 图标 hover 变色

动效持续时间：
**120–200ms**

---

## **9. Banner（Hero）视觉规范**

企业官网 Banner 与科技风不同，它更：

* 简洁、严肃
* 文案清晰
* 背景为照片或浅色渐变
* 大标题通常是一句话定位（Slogan）

规范：

* 标题最大 48–60px
* 副标题 18–20px
* CTA 两个（主 + 次）
* 推荐左文右图布局
* 图通常是企业场景、产品示意、建筑照片

---

## **10. 内容模块（Sections）**

企业官网核心模块：

1. **公司介绍（About）**
2. **业务范围（Business / Services）**
3. **成功案例（Case Studies）**
4. **公司优势（Why Us）**
5. **合作伙伴（Partners）**
6. **团队介绍（Team）**
7. **公司新闻（News）**
8. **联系我们（Contact）**

模块结构要求：

* 标题（大号）
* 子标题/Intro（中号）
* 图文布局（左右交错）
* 信息重点清晰（3–4 条为佳）
* 尽量严谨，不要营销式浮夸写法

---

## **11. 响应式规范（Responsive Guideline）**

### 桌面端

* 大量留白
* 宽空间排版
* 图文配合
* 内容对齐一致性非常重要

### 移动端

* 单列排版
* 图文上下排列
* 字号缩小
* 按钮全宽
* CTA 要固定区域展示

---

## **12. AI 生成页面的 Prompt 规范**

在用 AI 做页面时，加入这条指导，能让 AI 输出稳定职业风格：

```
请遵循企业级官网（Corporate Professional Style）设计规范：
1. 色彩：深蓝主色、灰色中性色为主视觉基调
2. 布局：12 列网格，大留白，信息结构清晰
3. 字体：中文思源黑体，英文 Inter
4. Banner：左文右图，专业、简洁、可信赖
5. 卡片：白底 + 灰边 + 轻阴影
6. 按钮：深蓝主按钮 + 白色次按钮
7. 图标：线性、简洁、商务
8. 动效轻度，强调稳重性
9. 整体视觉基调偏向大公司职业风格
```
