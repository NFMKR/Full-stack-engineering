# 🧭 **科技风 Web 官网 · 设计指南（Design Guideline – Tech Corporate Website）**

---

## **1. 品牌定位（Brand Positioning）**

本设计指南适用于科技型企业官网，包括 AI、云计算、数据技术、互联网应用、自动化等方向。
整体设计风格要求体现以下品牌特性：

* **高科技（Techy）**：未来感、数字化视觉、细节精致
* **专业（Professional）**：可信赖、逻辑严谨、布局规范
* **现代（Modern）**：大留白、克制配色、系统化网格
* **高端（Premium）**：统一视觉语言、高辨识度品牌形象

官网面向对象包括企业客户、合作伙伴、投资机构、公众用户。

---

## **2. 视觉基调（Visual Tone）**

科技风官网需呈现出：

* 轻量、干净的 UI 元素
* 线条、光影、渐变、网格等科技视觉语言
* 大留白 & 平衡布局
* 节奏分明的滚动展示（Landing Page 常见结构）

视觉目标：

> **突出科技感，而不过度炫技；专业可信，而不显沉重。**

---

## **3. 色彩系统（Color System）**

### **3.1 主色（Primary Color）**

科技蓝（Tech Blue）或自定义品牌色：

* `#165DFF`（科技蓝）
* 作用：重要按钮、主标题、图标高亮、品牌区域

### **3.2 辅助色（Secondary Colors）**

用于科技渐变、装饰、高光：

* 青绿色：`#36CFC9`
* 紫色：`#6A5BFF`

常用渐变：

```
linear-gradient(135deg, #165DFF 0%, #36CFC9 100%);
linear-gradient(135deg, #4B6CFF 0%, #6A5BFF 100%);
```

### **3.3 中性色（Neutral Colors）**

| Token    | 值       | 用途   |
| -------- | ------- | ---- |
| Gray-900 | #0A0A0A | 标题深色 |
| Gray-700 | #2F2F2F | 副标题  |
| Gray-500 | #888888 | 次要文字 |
| Gray-300 | #E0E0E0 | 分割线  |
| Gray-100 | #F7F8FA | 背景色  |

### **3.4 功能色（Semantic Colors）**

* Success: `#00C853`
* Warning: `#FFB300`
* Error: `#FF3B30`

---

## **4. 网格系统（Grid System）**

科技风官网建议采用：

* **12 列网格**
* 最大页面宽度：1440px
* 栅格间距：24px

移动端：4 列网格，边距 16px

> 所有模块保持左右对齐，提高页面一致性与专业感。

---

## **5. 字体系统（Typography）**

### **5.1 字体选择**

* **中文：`HarmonyOS Sans`、`Alibaba PuHuiTi`、`Source Han Sans`**
* **英文：`Inter`、`Roboto`**

### **5.2 字体层级**

| 类型      | 字号      | 行高  | 用途            |
| ------- | ------- | --- | ------------- |
| H1      | 48–64px | 1.1 | Banner / 核心标题 |
| H2      | 32–40px | 1.2 | 模块标题          |
| H3      | 24–28px | 1.3 | 小标题           |
| Body L  | 18px    | 1.6 | 正文较大          |
| Body M  | 16px    | 1.6 | 标准正文          |
| Caption | 14px    | 1.6 | 注释说明          |

Banner 主标题通常使用大号字体 + 渐变文字。

---

## **6. 组件造型（Component Style）**

科技风组件强调：

* 统一的圆角体系（建议：**10px 或 12px**）
* 阴影柔和、光感清晰
* 轻量边框（1px / 2px）
* 线性渐变按钮

### **6.1 按钮（Button）**

**主按钮（Primary）**

```
背景：科技渐变
颜色：#fff
圆角：12px
高度：48px
字体：16px SemiBold
阴影：0 4px 12px rgba(22,93,255,0.25)
hover：增强亮度
active：略微压低亮度
```

**次按钮（Secondary）**

```
背景：透明
边框：1px 主色
颜色：主色
hover：主色轻微渐变
```

---

## **7. Icon风格（Icons）**

科技风图标特点：

* 线性图标（2px stroked）
* 使用蓝色或灰色为主色
* 图标尽量扁平、避免复杂阴影

推荐库：
✔ Remix Icon
✔ Lucide
✔ Carbon Icons

---

## **8. 卡片与容器（Card / Container）**

科技风卡片视觉：

* 背景：`#FFFFFF` 或轻渐变
* 圆角：12px
* 阴影：`0 8px 20px rgba(0,0,0,0.06)`
* Hover：轻微浮起 + 边框高亮

---

## **9. 动效（Motion Guidelines）**

科技官网动效应遵循：

* **轻、柔、短，不要喧宾夺主**
* 使用 GPU 加速属性（transform / opacity）
* 时长 160ms–260ms

常见动效：

* Hero Banner：渐入 + 微缩放
* 滚动触发（Scroll Reveal）
* 按钮 hover：亮度增强
* Icon hover：描边亮色变化

---

## **10. Banner 区域设计规范（Hero Section）**

科技官网最重要的区域。

### 必须要素：

* 大标题（科技渐变文字）
* 副标题（简短、高价值）
* 两个主 CTA 按钮
* 背景视觉：科技线框 / 光粒子 / 抽象几何 / 数字人
* 布局偏左，右侧用于视觉图形

### 推荐元素：

* 动态粒子背景
* 科技蓝渐变光照
* 模糊光晕（Blured Glow）

---

## **11. 内容模块规范（Content Blocks）**

常见 3 模块结构：

1. **价值价值点（Value Proposition）**
   3–6 个图文矩形块，卡片 hover 放大

2. **功能模块介绍（Features）**
   左图右文 / 右图左文

3. **数据展示（Stats）**
   大号数字 + 副标题，表达实力

4. **案例（Cases）**
   滚动横向轮播，可视差

5. **联系我们（Contact）**
   CTA 强化 + 表单 + 公司地址

---

## **12. 整体交互风格（Interaction Rules）**

* 鼠标 hover 必须有明显反馈（光亮/边框/缩放）
* 滚动要有节奏感（分段 reveal）
* 注意响应式（三栏 → 两栏 → 单栏）
* 所有按钮至少有 *hover / active / disabled* 三态
* 动效不能太科技炫技，**要稳、要干净**

---

## **13. AI 生成（Prompt 规则）**

当让 AI 生成页面时，请加入以下指令：

```
请遵循我定义的 Web 科技风设计系统：  
1. 色彩：主色 #165DFF，辅色 #36CFC9 紫色 #6A5BFF  
2. 字体体系：Inter + 思源黑体  
3. 圆角：12px  
4. 按钮：科技渐变，带轻度阴影  
5. Hero 采用科技粒子风格背景  
6. 布局为左文右图  
7. 所有组件间距基于 8px 原子单位  
8. 图标使用线性风格  
9. 页面整体保持科技风、现代、专业  
```

你只需要一次告诉 AI，其后整个产品都会保持统一视觉效果。
